Endian: Little Endian unless specified.

----------------------------------------------------------------

Offset: Description
  0000: 16 bytes: header
  0010: DWORD: offset to data, usually 0x400.
  0014: DWORD, one of the following: [2, 3, 4]
  0018: DWORD: key_slot. Can only be 1.
  001C: BYTE[0x10]: Decryptor test data (validation only)
  002C: BYTE[0x10]: File Key
  ....: Padding
  ????: Encrypted body (*0x0010)
