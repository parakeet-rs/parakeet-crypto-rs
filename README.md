# parakeet-crypto-rs

使用 Rust 重新实现 [Parakeet][project_parakeet] 所支持的算法。

## 目前已实现的算法

- QMCv1 (static_map)
- QMCv2 (map / rc4)

## 命令行调用

你可以在项目百科查看[命令行调用][wiki_cli]相关的帮助内容。
